# WhackAMoleGame

## About
A simple Java-based Whack-A-Mole game created for our Programming 2 team project.  
Players try to click on moles that randomly appear in holes before they disappear!

## Class Structure
- **WhackAMoleGame** (Jacob Clark)  
  Main class that runs the game loop and interacts with other classes.
- **GameBoard** (Nathaniel Johnson)  
  Manages the board, mole positions, and user interactions.
- **ScoreTracker** (Susan Rogers)  
  Tracks player score and handles hits/misses.
- **Mole** (Lindsey Lear)  
  Represents an individual mole and its behavior.
- **GameTimer / DifficultyManager** (Danielle Kemp)  
  Tracks game time and adjusts difficulty as the game progresses.
